Features

  • ABN System – Automated Bot Notifications for server updates and moderation.
  • Moderation – Ban, kick, mute, warn users with easy commands.
  • Logging – Track message edits, deletes, joins, and leaves.
  • Fun Commands – Memes, jokes, 8ball, and more for your community.

Example Commands

  • /help – Show all commands.
  • /abn add [text] – Add an alert notification.
  • /kick @user – Kick a member.
  • /ban @user – Ban a member.
